## 功能简介
一个简易的四则计算器

## 运行方式
1. 命令行运行
	```bash
	$ ./go.sh
	```
2. 打开Browser  
稍等一会,等一下上述过程启动;然后浏览器访问
`http://127.0.0.1:5000/calc.php`
